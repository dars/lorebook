## MODIFIED Requirements

### Requirement: 新增角色（簡化版）
App SHALL 提供面向非專業玩家的引導式建角流程（基本〔含種族〕 / 職業 / 背景 / 能力值 / 技能與熟練 / 確認），職業·背景·種族選取後固定顯示敘述與重點提示，依 D&D 2024 規則計算衍生數值，建立後加入名冊並設為當前角色。

#### Scenario: 進入建角
- **WHEN** 使用者於角色選擇畫面點「新增角色」
- **THEN** 進入引導式建角流程，第一步為基本（角色圖佔位、名稱、陣營、種族，等級為 1）

#### Scenario: 選擇種族 / 職業 / 背景（含說明）
- **WHEN** 使用者於種族、職業或背景的選擇區
- **THEN** 列出精選選項供選擇
- **THEN** 選取後固定顯示該選項的敘述與重點提示（重點提示反映其機制效果）
- **THEN** 名稱、種族、職業、背景未備妥時不可進入下一步

#### Scenario: 能力值方式切換
- **WHEN** 能力值步驟顯示
- **THEN** 提供標準陣列／購點／擲骰三種方式切換，預設為標準陣列並套用建議配置
- **THEN** 於頁首說明已套用的背景能力加值
- **THEN** 本頁顯示六角屬性圖（實際分數），最終值為基礎值加背景加值且不超過 20

#### Scenario: 標準陣列
- **WHEN** 方式為標準陣列
- **THEN** 每能力以值下拉自 15/14/13/12/10/8 指派，選取已被佔用的值時與該能力對調
- **THEN** 顯示剩餘可分配數，並提供套用建議與清空
- **THEN** 六值皆指派（為該六值之排列）後方可續行

#### Scenario: 購點
- **WHEN** 方式為購點
- **THEN** 每能力以值下拉於 8–15 選取，顯示各值累計點數，剩餘點數買不起的值不可選
- **THEN** 顯示剩餘點數（上限 27）與重置；可重複值
- **THEN** 總花費不超過 27 點

#### Scenario: 擲骰
- **WHEN** 方式為擲骰
- **THEN** 每能力以數字輸入框手動填入（App 不代擲），夾於合理範圍且可重複、可低於 8
- **THEN** 六項皆填妥後方可續行

#### Scenario: 確認（角色預覽）
- **WHEN** 確認步驟顯示
- **THEN** 由上而下顯示：頭部（名稱與種族·職業·背景·陣營）、戰鬥數值（HP/AC/先攻/被動察覺/熟練，施法職業另含施法DC·命中）、六角屬性圖、屬性值
- **THEN** 豁免（含加值）與技能（含加值）以可收合區段顯示，預設收起
- **THEN** 熟練的豁免與技能以金色標示

#### Scenario: 技能與熟練
- **WHEN** 技能步驟顯示
- **THEN** 由職業可選技能清單中選取指定數量的技能熟練
- **THEN** 背景固定技能與職業固定豁免（2 項）自動帶入並唯讀顯示
- **THEN** 未選滿職業技能數量時不可續行

#### Scenario: 確認並建立
- **WHEN** 使用者於確認步驟點「建立」
- **THEN** 依輸入建立角色，並自動計算衍生數值（修正值、熟練加值、HP、AC、先攻、被動察覺、施法 DC/命中、依熟練的豁免與技能、種族速度）
- **THEN** 新角色加入名冊、設為當前角色，導航至主畫面
