## Context

`checks_section.dart`：分頁(高42) + 加值橫幅(數字44) + 三種網格。能力/豁免為 2 欄 `CompactStatRow`（內距 14/12、CN14+EN10+值17）；技能依屬性分組（群組標頭 + 2 欄格），共 18 項偏長。本變更純調密度與技能分頁呈現，不動互動。

## Goals / Non-Goals

**Goals:**
- 三個分頁的項目格更密、橫幅/分頁更精簡。
- 技能分頁在維持「依屬性分組」下更緊湊、好掃。

**Non-Goals:**
- 互動行為（分頁切換、點選顯示加值、再點取消、切頁清除）不變。
- 不加優勢/劣勢、不加熟練篩選/排序。
- 不動收合與頁面階層。

## Decisions

### 1. `CompactStatRow` 精簡
- 內距 `14/12` → `12/8`；CN 14→13、EN 維持 10、修正值 17→16。
- 選取/熟練的金色強調與邊框維持。
- 影響三個分頁（共用元件），一致變密。

### 2. 版面順序：加值橫幅在上、選單在下
- 順序改為 **加值橫幅 → 分頁選單 → 網格**（原為 分頁 → 橫幅 → 網格）；加值結果置頂最醒目，選單在下作為網格篩選。
- 加值數字 44 → 32（須容進固定高度）。
- **橫幅固定高度**：以空狀態高度為準（`height: 60`，取代 `minHeight`），有值/無值同高，內容垂直置中（`alignment: centerLeft` + 滿版 Row 維持右側數字對齊），**點選顯示加值時不造成版面跳動**。

### 2b. 分頁改無框底線式
- 取代原本的外框分段控制（bordered segmented box）。
- 三個分頁等寬並排，各為「文字 + 底線指示」：選取者金色粗體 + 金色底線；未選取灰字 + 淡色底線（形成連續基準線）。
- 無外框、無背景填色，與整頁扁平調性一致。

### 3. 技能分頁：分組保留、更緊湊
- 維持依屬性分組（STR/DEX/CON/INT/WIS/CHA）與群組標頭（小圓點 + 屬性名）。
- 群組底距 10→8、群組標頭下距 6→4、格列距 6→維持或微縮。
- 技能格沿用精簡後的 `CompactStatRow`（CN + EN + 值），2 欄維持（比單欄省空間）。

## Risks / Trade-offs

- **[CompactStatRow 共用]** → 改它會一致影響能力/豁免/技能（預期一致變密，正向）。
- **[更密的取捨]** → 點擊目標仍需夠大；內距 12/8 仍可點，金色選取回饋清楚。
- **[技能仍分組]** → 不改成字母排序；分組對應加值更實用，僅收緊間距。
